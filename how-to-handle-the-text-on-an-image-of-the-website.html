How to handle the text on an image of the website?

Yathin Babu

Placing the text on a image, for example while designing a header section in website, was one of the most frustrating thing I used to come across when I was new to the website development.

I used to edit the image directly to get enough contrast between image and text.

Then, after spending some time on Google I came across a technique called background overlay used in WordPress development.

It looks some thing like this

Irrespective of the device width, the text got sufficient contrast with respect to the image.

In CSS, we can achieve similar effect using background property on the image.

.image { background: linear-gradient(0deg,rgba(39, 151, 216,0.5),rgba(39, 151, 216,0.5)), url('images/image.jpg'); }
