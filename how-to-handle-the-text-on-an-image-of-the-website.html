How to handle the text on an image of the website?

Yathin Babu

While designing a website, one of the most common and frustrating thing we come across is placing a text at some random position on an image. It looks awesome if have the sufficient contrast between the image and the text, but it feels like a nightmare if we don't. At the time when I first noticed this problem, I used to edit the image itself to make the text look good on it. :(

One of the use-case is having a slider on the landing page of a website. A slider on a website is a changing content area with different "slides" of visual or information. The slides may include a large image and they may also include text and button overlaid on top.

Over the years of designing and developing websites, I came to know about 2 methods to overcome this problem which we be discussing below.

Background overlay

Background overlay is a feature available as a property in Wordpress development. But in CSS it's not available as a property, but by using a code similar to below on a image, we can achieve a similar effect.

.image {
background: linear-gradient(0deg,rgba(39, 151, 216,0.5),rgba(39, 151, 216,0.5)), url('images/home.jpg');
}

After applying the class mentioned above on an image of the website, we will get the result similar to this.

Setting background color for text

As the heading says, by setting the background color for the text we can overcome the contrast problem between an image and the text in front of it. I really don't recommend this method, as the background of the text blocks some parts of the image and looks awkward.

After applying the background color to the text, we will get the something similar to this.
