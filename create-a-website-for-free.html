Create a website for free

Yathin Babu

Creating a website is fun. If you are thinking of getting a job in IT industry or already working, then having a portfolio website is a must. In this article you will be walking through the ways you to create a static portfolio website for free.

GitHub Pages

GitHub is one of the world's largest Git repository hosting service. GitHub Pages also offers us to create static website for free. You will get one site per GitHub account and organization, and unlimited project sites.

Follow the steps below to create one for yourself:

  • Sign up to GitHub and create a new repository named username.github.io, where username is your GitHub username ( or organization name ) on GitHub. If the first part of the repository doesn't exactly match your username, it won't work, so make sure to get it right.

  • Download GitHub Desktop, it is a great way to use Git and GitHub on macOS and Windows.

  • After finishing the installation, head back to GitHub and refresh the page. Click the "setup up in Desktop" button. When the GitHub Desktop app opens, save the project. If the app doesn't open, launch it and clone the repository from the app.

  • Clone this repository, which hosts a responsive personnel portfolio website and copy paste the files to your repository.

  • Open your favorite text editor and edit the files you just downloaded as per your requirements.

  • Enter the GitHub Desktop app, commit your changes, and press the publish button.

  • And you're done. Open the browser and go to https://username.github.io.

Netlify

Netlify is a web developer platform that multiplies productivity.

  • Clone this repository, which hosts a responsive personnel portfolio website.

  • Open your favorite text editor and edit the files you just downloaded as per your requirements.

  • Sign up to Netlify.

  • Drag and drop the site folder you downloaded before inside the box with text Want to deploy a new site without connecting to Git? Drag and drop your site folder here.

  • Within few seconds the files will get uploaded and the site is live, the site url will be automatically generated. To change the url follow the steps:

    • Go to the Site settings

    • Under Site details, go to Site information and click "Change site name" button

      . Check weather the site name you want is available and save.

  • And you're done. Open the browser and go to https://sitename.netlify.app.
